# Learning Paths

Learning paths will combine Academy modules into goal-oriented sequences without
duplicating curriculum content.

Planned learning paths include:

- Quantum Developer
- Quantum Software Engineer
- Quantum Machine Learning Engineer
- Quantum Researcher
- Quantum Security Specialist
- Quantum Technology Leader

Each learning path will define:

- Target audience
- Expected background
- Required modules
- Optional modules
- Practical projects
- Assessment requirements
- Recommended next steps

Status: Foundation.
