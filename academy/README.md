# Quantum Computing Academy

The Quantum Computing Academy is the educational core of the Quantum Computing
Lab project.

It provides a structured and progressive learning environment designed to guide
learners from quantum computing fundamentals to advanced engineering and
research activities.

## Academy Components

| Component | Purpose |
|---|---|
| Curriculum | Progressive educational content organized by level |
| Learning Model | Educational structure, assessment and progression rules |
| Learning Paths | Role-based and objective-based study paths |
| Assessments | Quizzes, practical evaluations and final assessments |
| Certification | Future open-source completion and achievement framework |
| Instructors | Resources and guidance for educators and contributors |
| Shared Resources | Common assets, terminology and references |
| Templates | Standard structures for modules and educational content |
| Governance | Rules and processes for maintaining educational quality |
| Roadmap | Planned Academy milestones and curriculum development |

## Learning Model

The Academy's educational rules are defined in the
[Academy Learning Model](learning-model/README.md).

## Curriculum Levels

The Academy curriculum is organized into five progressive levels:

1. Beginner
2. Intermediate
3. Advanced
4. Expert
5. Research

The first active curriculum is the
[Beginner Track](curriculum/beginner/README.md).

## Educational Approach

The Academy combines:

- Conceptual explanation
- Essential mathematics
- Visual learning
- Practical Python development
- Interactive notebooks
- Exercises and challenges
- Framework comparison
- Simulator and hardware experimentation
- Reproducible research practices

## Current Status

The Academy is currently in the foundation phase.

The structure, learning model, templates and governance standards are being
established before full curriculum development begins.
