# Shared Academy Resources

This area contains resources used by multiple Academy tracks or modules.

## Structure

| Directory | Purpose |
|---|---|
| `assets/` | Shared images, diagrams and visual material |
| `bibliography/` | Structured bibliographic resources |
| `glossary/` | Common quantum computing terminology |
| `references/` | Shared books, papers and external resources |

Resources specific to a single module should remain inside that module rather
than being placed here.
