# Academy Curriculum

The curriculum provides the primary structured learning sequence of the Quantum
Computing Academy.

## Levels

| Level | Primary Focus | Status |
|---|---|---|
| Beginner | Foundations, mathematics, qubits, gates and measurement | Foundation |
| Intermediate | Circuits, algorithms, noise and practical development | Planned |
| Advanced | Optimization, machine learning, error correction and hardware | Planned |
| Expert | Architecture, benchmarking and advanced engineering | Planned |
| Research | Paper reproduction and original investigation | Planned |

## Curriculum Principles

Curriculum content should:

- Match the intended learner level
- Define clear objectives and prerequisites
- Introduce concepts before framework-specific implementation
- Include only the mathematics necessary for the learning objective
- Separate theory, exercises and solutions
- Provide reproducible code and execution instructions
- Cite authoritative references where appropriate
- Clearly communicate limitations and assumptions
