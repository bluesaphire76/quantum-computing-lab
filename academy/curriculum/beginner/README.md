# Beginner Track

The Beginner Track introduces the core concepts required to understand and
develop basic quantum computing applications.

No previous quantum computing or quantum mechanics experience is required.

## Track Objectives

By completing this track, learners should be able to:

- Explain the difference between classical and quantum computation
- Understand the mathematical representation of basic quantum states
- Describe superposition, interference, measurement and entanglement
- Work with single-qubit and two-qubit states
- Understand the principal single-qubit quantum gates
- Build and simulate introductory quantum circuits
- Create and analyze a Bell state
- Use a modern Python-based quantum development environment

## Estimated Effort

The complete Beginner Track is expected to require approximately 20–30 hours,
depending on the learner's mathematical and programming background.

## Prerequisites

- Basic familiarity with Python
- Basic high-school algebra
- Curiosity about quantum technologies

Previous knowledge of quantum mechanics is not required.
