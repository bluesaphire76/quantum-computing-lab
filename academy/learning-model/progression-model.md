# Progression Model

The progression model defines how learners move through modules, tracks and
curriculum levels.

## Module Progression

A module may have one of the following states:

- Not Started
- In Progress
- Practice Required
- Completed
- Mastered

## Completion

A module is Completed when the learner:

- Reviews the required theory
- Completes mandatory notebook activities
- Completes mandatory exercises
- Meets the assessment threshold
- Attempts the module challenge
- Can reproduce the required practical results

## Mastery

A module may be considered Mastered when the learner additionally:

- Completes the challenge independently
- Explains the result accurately
- Identifies limitations and assumptions
- Applies the concept in a new context
- Completes optional advanced activities

## Track Progression

A curriculum track should define:

- Required modules
- Optional modules
- Required practical projects
- Final assessment
- Completion criteria

## Curriculum Level Progression

### Beginner to Intermediate

The learner should be able to:

- Explain fundamental quantum concepts
- Use basic state notation
- Build and simulate simple circuits
- Interpret measurement results
- Work with basic multi-qubit systems
- Create and analyze a Bell state

### Intermediate to Advanced

The learner should be able to:

- Implement foundational algorithms
- Analyze transpilation and noise
- Compare simulators and backends
- Develop multi-step quantum experiments
- Interpret hardware execution results

### Advanced to Expert

The learner should be able to:

- Design hybrid quantum-classical workflows
- Evaluate optimization and machine learning techniques
- Apply error mitigation or correction concepts
- Make hardware-aware engineering decisions
- Perform meaningful benchmarking

### Expert to Research

The learner should be able to:

- Formulate a research question
- Review primary literature
- Design a reproducible methodology
- Analyze uncertainty and limitations
- Communicate results clearly

## Flexible Progression

The Academy recommends sequential learning but may allow learners to skip
modules when they can demonstrate the required prerequisite knowledge.

Learning paths may reuse curriculum modules without duplicating content.
