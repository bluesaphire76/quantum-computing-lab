# Assessment Model

The Academy assessment model verifies whether learners have achieved the stated
learning objectives.

## Assessment Types

### Knowledge Check

Purpose:

- Confirm immediate understanding
- Identify misconceptions early
- Provide rapid feedback

Typical format:

- Multiple choice
- True or false
- Short answer
- Concept matching

Knowledge checks are formative and do not determine module completion alone.

### Exercise

Purpose:

- Reinforce a specific concept or implementation skill

Typical format:

- Mathematical calculation
- Code completion
- Circuit modification
- Result interpretation
- Debugging task

### Practical Laboratory

Purpose:

- Validate the learner's ability to execute and interpret a complete experiment

Typical evidence:

- Working code
- Notebook output
- Analysis
- Reproducibility instructions

### Module Challenge

Purpose:

- Combine multiple objectives in a less-guided task

A challenge should:

- Define the required outcome
- Avoid giving a complete implementation
- Include clear validation criteria
- Allow more than one valid solution where appropriate

### Module Assessment

Purpose:

- Verify achievement of the module's primary learning objectives

A module assessment may combine:

- Knowledge questions
- Mathematical tasks
- Coding tasks
- Interpretation tasks
- Practical challenge

### Track Assessment

Purpose:

- Verify integrated competence across multiple modules

A track assessment should include a practical project requiring concepts from
several modules.

## Assessment Categories

Assessment should cover an appropriate combination of:

- Conceptual understanding
- Mathematical understanding
- Implementation
- Experimentation
- Interpretation
- Reproducibility
- Critical evaluation

## Passing Standard

Unless a module defines stricter requirements, recommended completion requires:

- At least 70% of knowledge-based assessment points
- All mandatory practical tasks completed
- Module challenge attempted
- No unresolved critical conceptual errors
- Reproducible execution of required code

## Solution Policy

Solutions must:

- Be stored separately from learner-facing exercises
- Explain reasoning, not only provide final answers
- Identify common mistakes
- Include validation steps
- Avoid being automatically exposed in the primary learning flow

## Assessment Integrity

The Academy does not currently provide accredited examinations.

Assessments are intended for:

- Self-evaluation
- Guided instruction
- Contributor validation
- Future open-source completion records
