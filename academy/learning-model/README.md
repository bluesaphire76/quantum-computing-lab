# Academy Learning Model

The Quantum Computing Academy learning model defines how educational content is
designed, structured, delivered, assessed and completed.

Its purpose is to ensure that every module provides a consistent, progressive
and technically accurate learning experience.

## Core Principles

Academy learning should be:

- Progressive
- Practical
- Technically accurate
- Reproducible
- Framework-aware but vendor-neutral
- Accessible to the intended learner level
- Explicit about prerequisites and limitations
- Focused on understanding before implementation

## Learning Sequence

The recommended sequence within a module is:

1. Orientation
2. Learning objectives
3. Prerequisite check
4. Conceptual theory
5. Essential mathematics
6. Visual explanation
7. Guided implementation
8. Interactive experimentation
9. Exercises
10. Challenge
11. Assessment
12. Reflection and next steps

## Learning Levels

The Academy contains five curriculum levels:

| Level | Primary Outcome |
|---|---|
| Beginner | Understand and implement foundational quantum concepts |
| Intermediate | Build and analyze practical quantum algorithms and circuits |
| Advanced | Develop applied hybrid and hardware-aware solutions |
| Expert | Engineer, benchmark and architect advanced quantum systems |
| Research | Conduct reproducible quantum computing investigations |
